GLOSARIO

  • Github: Es una plataforma en la nube que permite almacenar, compartir y trabajar en proyectos de software.
  • git push: Se utiliza para comunicar con otro repositorio, calcular lo que tu base de datos local tiene que la remota no tiene, y luego subir (push) la diferencia al otro repositorio.
  • git pull: Se emplea para extraer y descargar contenido desde un repositorio remoto y actualizar al instante el repositorio local para reflejar ese contenido.
  • HTML: Es un lenguaje de marcado que se utiliza para estructurar y desplegar páginas web. Es un componente fundamental de la web y se encarga de definir el significado y la estructura del contenido de una página.
  • IA: Es un campo de la informática que se encarga de crear sistemas que pueden realizar tareas que normalmente requieren inteligencia humana.
  • Formulario: Es una herramienta que permite a los usuarios enviar datos a un servidor. Ya sea que estés registrándote en un sitio web, dejando un comentario o realizando una búsqueda, un formulario es el puente que conecta tu información con la aplicación web.
  • CSS: Es un lenguaje de programación que se utiliza para controlar la presentación de los elementos de una página web.
  • Clases: Es un grupo de elementos que tienen características similares o son iguales.
  • Floats: Propiedad de posicionamiento que coloca un elemento al lado izquierdo o derecho de su contenedor.
  • Repositorio: Es un lugar donde puedes almacenar el código, los archivos y el historial de revisiones de cada archivo. Los repositorios pueden contar con múltiples colaboradores y pueden ser públicos como privados.
  • Extensiones VSC: Las extensiones de la CLI de GitHub son comandos de la CLI de GitHub personalizados que cualquiera puede crear y usar.
  • Modelo de caja: El modelo de caja es un elemento con su propio ancho y alto, un padding que marca la distancia entre el elemento y su borde, un borde que representa el límite entre el elemento y su exterior, y un margen que establece una separación entre los elementos.
